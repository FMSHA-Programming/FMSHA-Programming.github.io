# Обязательные упражнения. Сентябрь 2021

1. Написать программу для вычисления расстояния между двумя точками на плоскости. Координаты точек вводить с клавиатуры, расстояние выводить в консоль с точностью до третьего знака после запятой. Координаты точек имеет тип `int`, расстояние между точками имеет тип `double`. *Указание: используйте функцию `sqrt` из библиотеки `math.h`*
2. Написать программу, которая считывает целое число с клавиатуры и выводит квадрат и куб этого числа в консоль. Вычисление квадрата и куба реализовать через функции, которые вызываются из функции `main`.
3. Написать программу для решения линейного уравнения `a*x + b = 0`. Коэффициенты уравнения имеют тип `int` и вводятся с клавиатуры. Результат решения выводить в консоль. Обрабатывать все возможные значения.
4. Написать программу для вывода в консоль первых `n` чисел Фибоначчи. Число `n` вводится с клавиатуры.
5. Напишите функцию `swap`, которая меняет местами значения двух переменных типа `int`, не используя для этого дополнительных переменных.
6. Написать функцию, которая принимает массив типа `double` и определяет является ли массив отсортированным.
7. Написать функцию, которая принимает строку (`char[]`) и символ (`char`) и определяет содержится ли символ в строке. Функия возвращает ноль, если символ не содержится в строке и любое ненулевое целое число в противном случае.
